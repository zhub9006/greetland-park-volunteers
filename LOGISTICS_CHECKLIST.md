# ✅ Outing Logistics Checklist

A comprehensive checklist for organising the Greetland Community Group's volunteer park outing. Work through each section as the date approaches.

---

## 📅 Phase 1: Planning (6–8 Weeks Before)

### Destination & Event
- [ ] Agree on the destination park / AONB (see README for options)
- [ ] Identify a specific volunteer event or activity to join
- [ ] Contact the park / organisation to confirm volunteer places available
- [ ] Get a confirmed date and meeting point
- [ ] Check the event is suitable for mixed abilities / group size
- [ ] Confirm whether pre-registration is required (many parks require it)
- [ ] Add event details to the repo README and Issue #1

### Sign-Ups
- [ ] Announce the outing to the Greetland Community Group
- [ ] Share the sign-up link (Issue #1) with members
- [ ] Set a sign-up deadline (suggest 2 weeks before the date)
- [ ] Keep a running count of confirmed attendees
- [ ] Note any special requirements (accessibility, dietary, etc.)

---

## 🚌 Phase 2: Transport (3–4 Weeks Before)

*(See also Issue #2 — Transport Arrangements)*

- [ ] Count total attendees needing transport
- [ ] Count total car seats offered by drivers
- [ ] Decide on transport method:
  - [ ] Car sharing (coordinate via Issue #2)
  - [ ] Hired minibus / coach
  - [ ] Public transport (train + bus)
  - [ ] Mix of options
- [ ] If hiring a minibus:
  - [ ] Get quotes from local hire companies
  - [ ] Confirm driver (must hold appropriate licence)
  - [ ] Collect contributions from attendees
- [ ] If car sharing:
  - [ ] Match passengers to drivers (Issue #2)
  - [ ] Share meeting point and departure time
- [ ] Plan the route and check for any road closures / events
- [ ] Identify car park at destination (some parks charge — check in advance)
- [ ] Share final transport plan with all attendees

---

## 🎒 Phase 3: Supplies & Equipment (1–2 Weeks Before)

*(See also Issue #3 — Supplies & Kit)*

### Personal Kit (each volunteer to bring)
- [ ] Waterproof jacket and trousers
- [ ] Sturdy walking boots (ankle support recommended)
- [ ] Warm layers (even in summer — weather changes quickly on moorland)
- [ ] Hat, gloves (for cold or wet days)
- [ ] Sun hat and sunscreen (for warm days)
- [ ] Packed lunch and plenty of water (at least 1.5 litres)
- [ ] Small rucksack
- [ ] Personal first aid kit (plasters, blister patches)
- [ ] Any personal medication (inhalers, EpiPens, etc.)
- [ ] Phone (fully charged) with emergency contacts saved

### Group Equipment (organiser / park may supply)
- [ ] **Confirm with the park what tools they will provide** (most provide tools for their own events)
- [ ] Litter pickers and bin bags (if doing a clean-up)
- [ ] Work gloves (some parks provide, worth bringing your own)
- [ ] High-visibility vests (check if required)
- [ ] Group first aid kit
- [ ] Map of the area / OS map
- [ ] Emergency contact list (all attendees)
- [ ] Printed participant list

### Optional / Nice to Have
- [ ] Binoculars (for wildlife watching)
- [ ] Camera or phone for photos
- [ ] Notebook and pen (for wildlife surveys)
- [ ] Snacks to share at break time
- [ ] Reusable coffee flask
- [ ] Trowel / small tools (if doing planting)

---

## 📣 Phase 4: Communication (1 Week Before)

- [ ] Send a reminder to all confirmed attendees
- [ ] Confirm final meeting time and departure point
- [ ] Share the park's address and postcode for sat-nav
- [ ] Share emergency contact number for the park ranger / event organiser
- [ ] Check the weather forecast and advise on appropriate clothing
- [ ] Remind everyone to bring their personal kit (see above)
- [ ] Share any specific instructions from the park (e.g. no dogs, parking info)
- [ ] Create a WhatsApp group or similar for day-of communication

---

## 🌤️ Phase 5: On the Day

### Before Leaving
- [ ] Check the weather — if severe, contact park to confirm event is still on
- [ ] Ensure all drivers have the correct address / postcode
- [ ] Do a final headcount at the departure point
- [ ] Collect any outstanding contributions (minibus, parking, etc.)

### At the Park
- [ ] Sign in with the park ranger / event organiser on arrival
- [ ] Attend any safety briefing provided
- [ ] Distribute any group equipment
- [ ] Take a group photo! 📸
- [ ] Ensure everyone knows the emergency procedures
- [ ] Agree a meeting point in case anyone gets separated

### During the Activity
- [ ] Keep an eye out for anyone struggling — look after each other
- [ ] Take regular breaks — don't push too hard
- [ ] Stay hydrated and eat regularly
- [ ] Report any incidents to the park ranger immediately

### Heading Home
- [ ] Return all borrowed tools and equipment to the park
- [ ] Do a headcount before departing
- [ ] Leave no litter — take everything with you
- [ ] Thank the park rangers / event organisers

---

## 🎉 Phase 6: After the Outing

- [ ] Share photos in the group / on social media (with permission)
- [ ] Send a thank-you message to the park / event organisers
- [ ] Gather feedback from attendees (what went well? what to improve?)
- [ ] Update this repo with notes for next time
- [ ] Close the relevant issues (or open new ones for the next outing!)
- [ ] Celebrate a brilliant day's work! 🌿
